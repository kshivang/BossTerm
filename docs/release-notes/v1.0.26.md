# Release Notes - v1.0.26

**Release Date:** 2025-12-10

## Highlights
This release focuses on visual polish and stability improvements, including better cursor positioning during scrolling and resize operations, enhanced window blur effects for macOS, and fixes for inline image rendering.

## New Features
- Add configurable login session for "Last login" message display (macOS)
- Implement BlurEffect API for window background with improved macOS traffic light button styling

## Bug Fixes
- Fix cursor staying at buffer position when scrolling through history
- Fix cursor positioning when terminal shrinks to prevent extra prompt on resize
- Fix inline images rendering fully regardless of cursor position
- Fix blur effect implementation using proper graphicsLayer and Modifier.blur()
- Fix traffic light buttons by removing blur effect from their layer

## Improvements
- Simplify image rendering to pure placement-based approach
- Remove legacy image placement fallback code
- Remove debug logging from IncrementalSnapshotBuilder and image rendering
- Auto-trigger JitPack build on release
- Remove duplicate environment approval for publish-libraries workflow

## Breaking Changes
None
