# Release Notes - v1.0.63

**Release Date:** 2025-12-28

## Highlights
This release adds callback support for tab close events in TabbedTerminal, enabling applications to respond when users close tabs.

## New Features
- **Tab Close Callback**: Added `onTabClose` callback parameter to `TabbedTerminal` that receives the index of the closed tab (#194)

## Breaking Changes
None
