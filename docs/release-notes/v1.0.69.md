# Release Notes - v1.0.69

**Release Date:** 2026-01-03

## Highlights
This release introduces automatic shell integration injection for command completion notifications, eliminating the need for manual shell configuration.

## New Features
- **Auto-inject Shell Integration**: Automatic shell integration for command completion notifications works out of the box without manual `.bashrc`/`.zshrc` configuration (#202)
  - Supports Zsh (ZDOTDIR hijacking), Bash (ENV), and Fish (XDG_DATA_DIRS)
  - Automatically sends OSC 133 sequences for command tracking (A=prompt, B=command start, D=exit code)
  - Handles macOS `/usr/bin/login` wrapper by detecting actual shell from `$SHELL`
  - Preserves user's existing shell configuration files
  - Can be toggled via `autoInjectShellIntegration` setting (default: true)
  - New Settings UI toggle under Notifications section
