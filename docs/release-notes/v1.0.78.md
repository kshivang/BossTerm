# Release Notes - v1.0.78

**Release Date:** 2026-01-07

## Highlights
This release adds comprehensive Version Control System (VCS) integration with context-aware Git and GitHub CLI menus, and enhances AI assistant command interception with auto-run functionality.

## New Features
- **Version Control Context Menu**: Added right-click context menu with Git and GitHub CLI operations (#227)
- **Context-Aware Git Menu**: Git menu items dynamically show actual branches, repository status, and working directory context
- **Git Checkout/Switch Submenus**: Interactive submenus displaying all available branches for quick checkout
- **GitHub CLI Context Awareness**: GitHub CLI menu shows warnings when repository is not configured
- **VCS Tool Installation Dialogs**: Added installation dialogs for Git and GitHub CLI with embedded terminal output
- **AI Command Auto-Run**: AI assistant commands now auto-execute after successful installation
- **Platform-Aware Install Commands**: VCS tool installation adapts to the user's platform and package manager

## Improvements
- **URL Opening Utility**: Refactored URL opening logic into shared `UrlOpener` utility for consistent cross-platform behavior
- **Working Directory Detection**: Improved process working directory detection by finding child process PID
- **Process Cleanup**: Enhanced process cleanup with nested try-finally blocks and timeout protection
- **AI Installation Dialog**: Deduplicated AI install dialog code and fixed process blocking issues
- **Installation Feedback**: Installation results now write to parent terminal for better visibility
- **Detection Timing**: Moved detection refresh to dialog dismiss to avoid race conditions
- **ANSI Color Output**: Fixed ANSI color output using proper echo command

## Bug Fixes
- **Process Timeout**: Added 2-second timeout to all `ProcessBuilder.waitFor()` calls to prevent hangs
- **VCS Install Commands**: Fixed VCS tools to use direct install commands instead of install dialogs
- **Process Resource Leak**: Added process cleanup to prevent resource leaks on timeout
- **Dialog Close Button**: Fixed close button display on successful installation instead of auto-close
- **Code Quality**: Addressed multiple code quality issues identified in review

## Documentation
- Added VCS tool installation API documentation to embedding docs
- Updated CLAUDE.md with VCS tool install feature details
- Clarified VCS tool API and settings behavior
- Added comprehensive AI Assistant Integration documentation

## Breaking Changes
None
