# Release Notes - v1.0.92

**Release Date:** 2026-02-20

## Highlights
This release introduces custom PlatformServices support, allowing developers to extend BossTerm's platform-specific behavior through a flexible service injection API. Examples have been updated to demonstrate this capability.

## New Features
- **Custom PlatformServices**: Allow adding custom platform services, enabling developers to override or extend platform-specific behavior in embedded and tabbed terminal integrations
- **PlatformServices demos**: Added demonstrations of custom PlatformServices in both the tabbed and embedded example projects

## Bug Fixes
- Fixed inconsistent `platformServices` naming across the codebase

## Improvements
- Added usage documentation and wiki pages for the Custom Platform Services API
- Updated README with a Custom PlatformServices example section

## Breaking Changes
None
