# Release Notes - v1.2.105

**Release Date:** 2026-06-05

## Highlights
Session sharing gains a live switch into remote mode, verified Cloudflare share links, and full viewer parity so remote viewers stay in sync with resizes, splits, focus changes, and tabs.

## New Features
- **Live remote-mode switch**: Toggle an active session into remote (shared) mode on the fly without restarting the session.

## Bug Fixes
- Cloudflare share links are now verified before being handed out, preventing dead or unreachable links.

## Improvements
- **Viewer parity**: Remote viewers now mirror the host across resizes, pane splits, focus changes, and tab switches for a consistent shared experience.

## Breaking Changes
None
