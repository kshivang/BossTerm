# Release Notes - v1.2.110

**Release Date:** 2026-06-08

## Highlights
This release improves the session sharing experience with one-click cloudflared installation on Linux and a new embedder hook that lets hosts fit their screen to the viewer.

## New Features
- **One-click cloudflared install on Linux**: Session sharing now offers a seamless, one-click install of cloudflared on Linux, removing manual setup steps (#286).
- **"Fit host to my screen" embedder hook**: Added an embedder hook for session sharing that allows fitting the host display to the viewer's screen (#285).
