# Release Notes - v1.2.117

**Release Date:** 2026-06-29

## Highlights
This release adds a `show_image` MCP tool and a global show/hide hotkey toggle, alongside a significant idle-CPU reduction that eliminates roughly a full core of wasted drain.

## New Features
- **MCP `show_image` tool** — display images directly through MCP, with split targeting fixed to the agent's own tab (#296, #300)
- **Global show/hide hotkey toggle** — a dedicated hotkey to bring the window in and out of focus (#299)

## Bug Fixes
- Fixed MCP split targeting so splits open in the agent's tab rather than the wrong one (#296, #300)

## Improvements
- **Hotkeys**: two-modifier default with a conflict warning when bindings collide (#299)
- **Performance**: eliminated ~1 full CPU core of idle drain caused by the hotkey CFRunLoop spin, and cut unnecessary blink repaints (#298)
- **Docs**: refreshed README with a new hero, plus documentation for session sharing and Remote MCP (#294)

## Breaking Changes
None
