# Release Notes - v1.2.118

**Release Date:** 2026-06-30

## Highlights
This release improves split-pane and tab title handling, adds BossConsole account sign-in to the Share menu, and delivers significant rendering performance gains.

## New Features
- **Sign In via BossConsole account** in the Share menu (#295)
- **Background-tab title updates** plus MCP split-pane and `get_active_tab` targeting (#301)

## Bug Fixes
- Stopped split-pane titles from landing on the root pane's chip; the window title now follows the focused pane (#304)

## Improvements
- Parked idle repaints, fixed a per-tab display leak, and cut render churn for better performance (#302)

## Breaking Changes
None
