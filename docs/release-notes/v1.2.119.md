# Release Notes - v1.2.119

**Release Date:** 2026-06-30

## Highlights
This release introduces a new default "BOSS Operator" terminal theme and palette, and lets in-shell tools automatically discover the host's MCP server via the `BOSS_MCP_SERVER` environment variable.

## New Features
- **MCP server discovery**: Expose `BOSS_MCP_SERVER` so in-shell tools automatically pick up the host's MCP server (#305).

## Bug Fixes
None

## Improvements
- **New default theme**: Added the BOSS Operator terminal theme and palette, now used as the default (#303).

## Breaking Changes
None
