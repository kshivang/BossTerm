# Release Notes - v1.2.127

**Release Date:** 2026-07-10

## Highlights
This release strengthens MCP multi-instance handling with polite auto-reattach and more reliable port registration for Claude Code, plus documentation improvements for downloads.

## New Features
- **Polite multi-instance auto-reattach**: MCP now uses an `/identity` probe to politely auto-reattach across multiple instances (#321).

## Bug Fixes
- **Correct port in Claude Code registrations**: MCP now bakes the configured port into Claude Code registrations instead of a fallback-walked one (#328).

## Improvements
- **Documentation**: README now links stable, always-latest download URLs (#322).

## Breaking Changes
None
