# Release Notes - v1.2.130

**Release Date:** 2026-07-12

## Highlights
This release focuses on stability, fixing a rendering crash when glyphs fall outside the canvas and a crash during MCP plugin hot-reload.

## Bug Fixes
- **Rendering:** Don't crash the window when a glyph lands outside the canvas (#333)
- **MCP:** Survive engine teardown after the host closes our classloader, which previously crashed on plugin hot-reload (#332)
