# Release Notes - v1.2.134

**Release Date:** 2026-07-22

## Highlights
This release adds inline image support via the Kitty and Sixel graphics protocols, and fixes a terminal freeze that occurred with large numbers of concurrent sessions.

## New Features
- **Graphics protocols**: Added support for the Kitty and Sixel image protocols, enabling inline image rendering in the terminal (#337).

## Bug Fixes
- Kept per-session blocking loops off the shared coroutine pools, resolving a terminal freeze that occurred at ~20 concurrent sessions (#338).

None
